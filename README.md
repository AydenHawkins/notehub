# Student Note Sharing Platform
**NoteHub** is a collaborative, course-specific file sharing and discussion tool designed for university students. It allows students to securely store, publish, and access academic notes while engaging with classmates in shared course environments.

## Project Overview
This platform fosters a connected academic community by enabling students to:
- Upload and manage personal academic notes.
- Share course-specific notes with peers.
- View, comment, and collaborate through class discussion boards.
- Explore a shared archive of public notes organized by semester and course.
- Manage course enrollments and track academic history.

In addition, platform administrators can:
- Monitor and manage user accounts.
- Moderate reported content.
- Oversee course listings and semester transitions.
